Nested loops in c in c

C programming allows to use one loop inside another loop. The following section shows a few examples to illustrate the concept.

Syntax of nested for :
for ( init; condition; increment ) {
for ( init; condition; increment ) {
statement(s);
}
statement(s);
}
Syntax of nested while :
while(condition) {
while(condition) {
statement(s);
}
statement(s);
}
Syntax of nested do-while :
do {
statement(s);
do {
statement(s);
}while( condition );
