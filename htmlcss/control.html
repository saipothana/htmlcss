Control statements in c

Types of control statements

  1. Decision making statements(if,if-else)
  2. Selection statements(switch-case)
  3. Iteration statements(for,while,do-while)
  4. Jump statements(break,continue,goto)

Decision making statements(if,if-else)

In decision making we have if,if-else,else-if,nested if...
if:  if is used to check when the condition is true only.

Syntax:
if(cond){
//sratements
}
Example
void main(){
int a=3;
if(a>2){
printf("true");
}


if-else:  It is used to check when the condition is two conditions
Syntax:
if(cond){
//sratements
}
else{
//statements
}
Example
void main(){
int a=1;
if(a>2){
printf("true");
}
else{
printf("false");
}


else-if:  It is used to check more than one conditions
Syntax:
if(cond){
//sratements
}
else if(cond){
}
else if(cond){
}
else{
//statements
}
Example
void main(){
int a=1;
if(a==1){
printf("a is 1");
}
else if(a==2){
printf("a is 2"); }
else{
printf("a is neither 2 nor 1");
}

Selection statements(switch-case)

Switch statement:
A switch statement is used for multiple way selections that will branch into different code segments based on the value of a variable or expression.
This expression or variable must be of integer data type.
Syntax:
switch(expression){
case 1:
//statements
break;
case 2:
//statements
break;
case 3:
//statements
break;

default:
//statements
}

Example:
void main( ) {
int day=5;
switch(day)
{
case 1:
printf("Sunday");
break;
case 2:
printf("Monday");
break;
case 3:
printf("Tuesday");
break;
case 4:
printf("Wednesday");
break;
case 5:
printf("Thursday");
break;
case 6:
printf("Friday");
break;
case 7:
printf("Saturday");
break;
default:
printf("Invalid choice");
}

Iteration statements(for,while,do-while)


Iteration statements are used to execute a particular set of instructions repeatedly until a particular condition is met or for a fixed number of iterations.
  • forloop
  • whileloop
  • do-whileloop

forloop:  it is used when we know the no.of iterations.It is entry control loop.
Syntax:
for(initialization; termination; increment/decrement) { //statements to be executed
}
Example:
void main(){
for(int i=1;i<=10;i++){
printf("hi");
}


Whileloop:  it is used when we don't know the no.of iterations.It is entry control loop.
Syntax:
while(cond){
//statements
}
Example:
void main(){
int i=1;
while(i<=10){
printf("hi");
i++}



do-Whileloop:  it is used when we don't know the no.of iterations.It is exit control loop.
Syntax:
do{
//statements
}while();
Example:
void main(){
int i=1;
do{
printf("hi");
i++}while(i<=10);



Jump statements(break,continue,goto)

break:  it is used to come out of the loop

continue:  it is used to skip the particular condition

goto:  It is used to goto the previous
