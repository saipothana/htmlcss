  • User defined functions(not mandatory)

Syntax of c

#include stdio.h
void main() {
printf("Hello World!");
}

line 1:  #include is a is a header file library that lets us work with input and output functions, such as printf() . Header files add functionality to C programs.

line 2:   void main() every thing is happened inside the main function..void means not return to anything it consists of opening { and closing curly braces }

line 3:   printf() is a function used to output/print text to the screen. In our example it will output "Hello World".
